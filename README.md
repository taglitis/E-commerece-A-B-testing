# E-commerece-A-B-testing

**Analyze A/B Test Result** is a project at Udacity Data Analysis Nanodegree program. In this project A/B hypothesis testing has been completed on the provided dataset. The goal of the project is to decide either an old page or a new page is better. In addition, for the comparison purposes, a regression approach was used. 

The python code was developed in Jupyter Notebook which is well documented and easy to follow.
